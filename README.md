# موقع شركة أميرة طوبار

موقع احترافي شبيه بموقع Apple لعرض وبيع منتجات شركة أميرة طوبار.

## المميزات

- تصميم احترافي شبيه بموقع Apple
- دعم كامل للغة العربية (RTL)
- قسم فيديو/فلاش في الصفحة الرئيسية
- أقسام المنتجات:
  - عطور فاخرة
  - منظفات عالية الجودة
  - خلاطات طعام احترافية
  - منتجات العناية بالبشرة والجسم
- نموذج اتصال مع ربط البريد الإلكتروني
- روابط وسائل التواصل الاجتماعي:
  - فيسبوك
  - تيليجرام
  - واتساب (زر عائم + رابط في صفحة الاتصال)

## الملفات

- `index.html` - الصفحة الرئيسية
- `styles.css` - ملف التنسيقات
- `script.js` - ملف JavaScript للتفاعلات

## إضافة فيديو للصفحة الرئيسية

لإضافة فيديو للصفحة الرئيسية:

1. ضع ملف الفيديو في نفس مجلد الموقع
2. اسم الملف: `hero-video.mp4`
3. إذا لم يكن الفيديو موجوداً، سيتم عرض خلفية متدرجة بدلاً منه
